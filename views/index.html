Image Search Abstraction Provider

Returns a JSON object containing a list of images with data.

Image Search Input

https://ssc-image.glitch.me/search + ?q=(search query)

https://ssc-image.glitch.me/search + ?q=(search query) + &off=(optional offset value)

Image Search Example URL

https://ssc-image.glitch.me/search?q=kendrick lamar

https://ssc-image.glitch.me/search?q=drake&off=5
